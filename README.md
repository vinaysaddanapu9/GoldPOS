# 💎 GoldPOS - Jewellery Billing System

A Python Tkinter-based desktop application for gold calculation and billing with thermal printer support.

## 🚀 Features
- Multi-row gold calculation
- Percentage purity support
- Optional gold rate calculation
- Print receipt (USB printer)
- Preview bill before printing
- Clean POS-style UI

## 🛠️ Tech Stack
- Python 3.11.5
- Tkinter
- ESC/POS Printing
